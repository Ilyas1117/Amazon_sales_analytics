Project Overview

This project focuses on analyzing Amazon sales data. The goal was to build a complete data pipeline — from cleaning and transforming raw data to creating dashboards for business insights.

The project workflow includes:

Data Collection: Used a dataset from Kaggle containing Amazon sales data.
Data Cleaning & Transformation: Handled missing values, removed anomalies, and prepared the dataset for normalization.
Normalization: Split a single CSV into multiple tables to create a normalized relational database structure.
PostgreSQL Integration: Loaded normalized data into PostgreSQL and created relationships and constraints to ensure data integrity.
Analytical Queries: Designed SQL queries for aggregation, filtering, and data preparation for visualization.
Visualization: Built dashboards in Power BI and Apache Superset, displaying key metrics, sales trends, category breakdowns, and regional performance.

Tools & Technologies

Database: PostgreSQL
Visualization: Power BI, Apache Superset
Data Processing: Python (pandas, numpy)
ETL & SQL: SQL for transformations, constraints, and analytics
